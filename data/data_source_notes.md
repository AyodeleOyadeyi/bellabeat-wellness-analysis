# Data Source Notes

## Dataset

This project uses the publicly available **FitBit Fitness Tracker Data** dataset made available by Mobius on Kaggle.

The dataset contains personal fitness tracker data from Fitbit users, including daily activity, steps, calories, sedentary minutes, and sleep records.

## Source

Dataset: FitBit Fitness Tracker Data  
Platform: Kaggle  
Publisher: Mobius  
License: CC0: Public Domain

## Files Used in This Analysis

The primary files used in this project were:

- Daily activity data
- Sleep data

The analysis focused on daily activity and sleep behavior, including:

- Daily steps
- Activity intensity minutes
- Sedentary time
- Calories
- Sleep duration
- Time in bed
- Sleep efficiency

## Notes on Raw Data

Raw data sheets were preserved inside the Excel workbook for transparency. Working copies were created for cleaning, transformation, analysis, and dashboard development.

The project includes the final Excel workbook rather than uploading every raw dataset file separately.

## Data Limitations

This dataset has several limitations:

- The sample size is small.
- The data was collected from Fitbit users, not Bellabeat users.
- Demographic variables such as age, gender, location, and health status were not available.
- The dataset covers a limited time period.
- Some users have activity data without matching sleep data.
- The analysis identifies associations and trends, but it does not establish causation.

These limitations should be considered when interpreting the findings and applying them to Bellabeat's marketing strategy.
